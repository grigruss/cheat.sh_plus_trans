Bash script for translating tips from [cheat.sh](http://cheat.sh/)

### Welcome

This script requires curl and [Translate Shell](https://github.com/soimort/translate-shell).

### Usage
```markdown
# Help tip about a command
$ cheat.sh commandname

# Find help tips
$ cheat.sh keyword
```
